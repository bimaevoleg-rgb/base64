# 🔐 Base64 Конвертер

Универсальный Base64 конвертер с веб-интерфейсом. Кодирование и декодирование текста и файлов прямо в браузере.

## Возможности

- **Текст ↔ Base64** — кодирование и декодирование текста с поддержкой UTF-8 (кириллица, эмодзи, иероглифы)
- **Файл ↔ Base64** — загрузка файлов через drag & drop, конвертация в Base64 и обратно
- **Drag & Drop** — перетаскивайте файлы для кодирования
- **Превью** — предпросмотр изображений и метаданных файлов
- **Скачивание** — сохранение декодированных файлов с выбором имени
- **100% клиентский** — все данные обрабатываются только в браузере, ничего не отправляется на сервер

## Файлы

| Файл | Описание |
|------|----------|
| `index.html` | Основное приложение — Base64 конвертер |
| `test.html` | Страница автоматических тестов (запускается в браузере) |
| `test_cli.js` | CLI тесты — запуск: `node test_cli.js` |

## Запуск

Откройте `index.html` в браузере:

```bash
# Локально
open index.html

# Или через простой сервер
python -m http.server 8080
```

## Тесты

**CLI:**
```bash
node test_cli.js
```

**Браузер:** Откройте `test.html`

## Технологии

- Чистый HTML/CSS/JavaScript, без зависимостей
- Web API: `TextEncoder`, `TextDecoder`, `btoa`, `atob`, `Blob`, `FileReader`
- Drag & Drop API
- CSS Custom Properties для тёмной темы
